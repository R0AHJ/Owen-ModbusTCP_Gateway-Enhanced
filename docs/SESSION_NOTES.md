### Реализованные улучшения

#### 1. Управление каналами
- **enable_channel** — включить канал для опроса
- **disable_channel** — отключить канал от опроса (без удаления из конфига)
- **get_channel_status** — показать статус всех каналов устройства
- Флаг `enabled: true/false` в конфигурации точек

#### 2. Валидация конфигурации
- Проверка пересечений OWEN-адресов (устройства на одной линии)
- Проверка пересечений Modbus-адресов (для одного Slave ID)
- Проверка выхода за допустимые диапазоны
- Текстовый отчёт с рекомендациями

#### 3. CLI-команды
```bash
owen_gateway config channel-status    # Статус каналов устройства
owen_gateway config channel-enable   # Включить канал
owen_gateway config channel-disable  # Отключить канал
owen_gateway config validate         # Валидация конфигурации
```

#### 4. Bash-меню (Русский интерфейс)
Файл: `deploy/linux/owen-gateway.sh` (733 строки)

Режимы работы:
- **Интерактивный** — цветное меню с выбором пунктов
- **Командный** — запуск с параметрами, например: `./owen-gateway.sh start`

Пункты меню:
1. Управление службой (start/stop/restart/status)
2. Инструменты конфигурации (просмотр, добавление, экспорт)
3. Управление каналами (enable/disable/status)
4. Проверка конфигурации (валидация)
5. Диагностика (зонд — проверка связности)
6. Системная информация (CPU, RAM, диски, сеть)
7. Резервное копирование
8. Просмотр логов (journalctl, tail)
